┌──────────────────────────────────────────────────────────────────────────────┐
│                            🛠️ TROUBLESHOOTING                                │
└──────────────────────────────────────────────────────────────────────────────┘

ISSUE: Connection Refused
   SYMPTOM: Can't connect Rojo plugin
   CHECK: Is Rojo server running?
   FIX: Run START_ROJO.bat or "rojo serve"
   VERIFY: Terminal shows "listening on port 34872"

ISSUE: Scripts Not Appearing
   SYMPTOM: No BloxVille folder in Studio
   CHECK: Did you click "Sync In"?
   FIX: Click "Connect" then "Sync In"
   VERIFY: Check Output window for errors

ISSUE: No Music Playing
   SYMPTOM: Silent when playing game
   CHECK: Is BackgroundMusic in SoundService?
   FIX: Check Output for errors, verify sound ID
   VERIFY: Look for "BackgroundMusic" sound object

ISSUE: No Welcome Screen
   SYMPTOM: Banner doesn't show
   CHECK: Is WelcomeScreen.lua synced?
   FIX: Check PlayerGui when playing
   VERIFY: Look for "WelcomeScreen" ScreenGui

ISSUE: Rojo Plugin Missing
   SYMPTOM: Can't find Rojo in Studio
   FIX: Run "rojo plugin install"
   VERIFY: Restart Roblox Studio
