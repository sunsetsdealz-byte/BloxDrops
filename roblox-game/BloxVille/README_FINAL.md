## 🐛 TROUBLESHOOTING

**Nothing shows up?**
- Check you copied files to correct folders
- Check Output (F9) for errors
- Make sure GameInitializer.lua is in ServerScriptService

**Map doesn't generate?**
- Check MapBuilder.lua is in ServerScriptService
- Check Config.lua is in ReplicatedStorage.Modules

**UI doesn't work?**
- Check all client scripts are in StarterPlayerScripts
- Check ClientInit.lua is there
