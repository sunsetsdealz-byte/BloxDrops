## 🚀 Setup Instructions

1. Create new Roblox place in Studio
2. Enable API Services (Studio Settings → Security)
3. Copy all scripts to ServerScriptService
4. Copy UI elements to StarterGui
5. Import models from `Assets/` folder
6. Configure game settings in `Config.lua`
7. Test in Studio, then publish
