   - Copy ALL 3 .lua files
   - Paste into StarterPlayer → StarterPlayerScripts

4. SETUP CONFIGURATION
   - In ReplicatedStorage, create folder: Modules
   - Copy Config.lua from Modules folder to ReplicatedStorage.Modules
   - In ReplicatedStorage, create folder: Events
   - In Events folder, create RemoteEvent: PlaySound
   - In ReplicatedStorage, create folder: Vehicles (leave empty)

5. TEST
   - Press F5
   - Map should generate
   - NPCs should spawn
   - Street lights should appear
   - HUD should show (top right)
   - Press TAB for menu

6. PUBLISH
   - Create icon/thumbnail

🆘 TROUBLESHOOTING
══════════════════════════════════════════════════════════════

PROBLEM: Map doesn't generate
SOLUTION: Check MapBuilder.lua in ServerScriptService
         Check Config.lua in ReplicatedStorage.Modules

PROBLEM: NPCs not spawning
SOLUTION: Wait 5 seconds after game starts
         Check NPCSystem.lua is running

PROBLEM: Street lights don't work
SOLUTION: Wait for night time (6 PM in-game)
         Check StreetLights.lua is running

PROBLEM: UI not showing
SOLUTION: Check MainUI.lua in StarterPlayerScripts
         Check leaderstats exist

PROBLEM: Sounds not playing
SOLUTION: Sounds only work in published game
         Check AmbientSounds.lua is running
